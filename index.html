Estrutura Básica do HTML

Subir

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags <html>, <head> e <body> e a instrução <!DOCTYPE> .

Falando Sobre Tags

Subir

Existem algumas tagas que geralmente são usadas quando vamos escrever o nosso documento html.
Temos a tag <p> ´que é utilizada para escrevermos paragrafo no documento html.
Temos a tag <strong> que deixa o texto em negrito.
Temos a tag <i> que deixa o texto em italico.
Temos a tag <u> que deixa o texto sublinhado.
Temos também a tag <img> que por sua vez, insere imagens no documento:
<img src="https://revistacarro.com.br/wp-content/uploads/2021/06/Fiat-Pulse_1.jpg" width="150">

- Atributo das Tags

Subir

Existem algumas tags que recebem atributos, ou seja, através de uma regra podemos determinar alguma regra para aquela tag, Ex:
<input type="text">
Esse "type" é um atributo da tag input, que através dele podemos determinar que a regra daquela input será text ou texto.

- Tags Extras

Subir

<sup> é utilizado para escrever um texto com letras menores acima das palavras ou caracteres.
titulo*

<blockquote> Tag usada para citações, obs, e etc...

Repellat fuga perferendis possimus.

<small>
Tag usada para criar textos pequenos, simples.

<font>
A tag foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto.

<del>
A tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.

<abbr title="">
É utilizada para fazer abreviações.

Listas

Subir

Existem dois tipos de listas, as ordenas "ol" e as não ordenadas "ul".
<ol>
<li></li>
</ol>

São lista que são sequencias por uma ordem.
  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3.1
    2. Item 3.2
<ul>
<li></li>
</ul>

São lista que não tem ordenamento, são marcadas pelos bullets points.
  • Item 1
  • Item 2

Links

Subir

O Elemento HTML <a> especifica as relações entre o documento atual e um recurso externo, ou seja, ele pode nos levar a um determinada pagina extern ou navegar no proprio site.
Estrutura:

<a href="http://dio.me" target="_blank" title="Clique aqui e confira">DIO</a>
